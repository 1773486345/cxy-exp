# 研究问题复位

> 状态：本文件定义下一阶段的研究边界，不提出新模型设计，也不授权改变现有模型行为。

## 原始研究问题

将预测模型中用于处理非平稳性的时序分解，转化为面向异常检测的分解。

这里的重点不是借用预测任务的误差目标，而是检验分解在异常检测中是否能保留并揭示异常证据。

## 核心问题

下一阶段只回答以下问题：

1. 分解后的每个结构部分是否保留异常信息。
2. 不同部分是否对不同异常具有专门性。
3. 分量异常证据是否互补。
4. 分量评分是否优于只使用最终总残差。

## 当前不研究

下列内容不属于当前阶段的研究范围：

- 下一点概率预测；
- 目标盲协议；
- 动态尺度校准；
- 测试时适应；
- adaptive cutoff；
- 动态路由；
- 新主干网络。

APD-CATCH v1/v2 的代码、结果和文档因此被视为旧探索线，保留但不扩展；其任何结果都不得自动并入新的分解研究主表。

## 基座与变更纪律

新阶段必须以原版 CATCH 的完整窗口重构协议为基座。与原版的比较必须保持同一协议语义，不能以旧探索线的 past-to-next-point 结果替代或混入该比较。

所有修改必须遵守“一次只改变一个变量”。每次比较都要保持其余训练数据、切分、评价、阈值与实现条件不变，并明确记录唯一变化项。未经人工确认，不实施任何代码、脚本、配置或结果目录改动。
